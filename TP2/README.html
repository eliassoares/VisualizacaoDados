Comentários sobre a infraestrutura do trabalho

Para melhor organizar o trabalho prático, o grupo optou pelo uso da ferramenta Bower. Esta ferramenta auxilia no gerenciamento das dependências front-end. Explicando de maneira simples, ela registra quais dependências foram instaladas no arquivo bower.json. Quando o projeto é compartilhado via repositório, em vez de se copiar os arquivos das dependências (tais como jquery ou d3.js), basta executar o comando bower install e todas elas serão baixadas.

No caso deste trabalho, incluímos os códigos das dependências na pasta compactada que acompanha este arquivo, todas instaladas dentro do diretório bower_components. Colocamos aqui este ccomentário apenas para informar do que se trata o Bower e explicar a organização dos diretórios.
