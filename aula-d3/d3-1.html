D3

  • Padrões web
  • Voltado a dados
  • As visualizações são imagens em formato SVG
  • Formatos: CSV e JSON
  • Livro: D3.js in Action
  • D3 te dá o poder de criar suas próprias técnicas e visualizações avançadas
  • Para gráficos simples existem bibliotecas mais convenientes

Passos da visualização

  1. Coletar e processar os dados
  2. Cria a estrutura da visualização
  3. Desenha a visualização

Example

        d3.selectAll("div")
          .data(someData)    // Data binding
          .enter()
          .append("div")
          .html("Wow")
          .append("span")
          .html("Even More Wow")
          .style("font-weight", "900")
